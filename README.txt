Stellen Sie vor der Installation sicher, dass Sie eine Datenbank für unsere App erstellt haben. 
Der Name unserer Datenbank ist chatten_db. 
Die Daten der Tabellen finden Sie in der Datei chatten_db.sql.
Nebenbei bemerkt: Das Passwort für die Standardkonten ist 123

Die Details der Files in Projekt

addfriend.php       --> die Freude Seite, wo der Benutzer seine Freude mit Email suchen kann.
chat.php            --> die Hauptseite bzw. Chat Seite, wo der Benutzer die Freundliste, das Chatfenster 
                        und die Buttons bwz. Logout, Delete Friend oder Block friend sehen kann.
connection.php      --> die Information über die Datenbanksverbindung.
findhomies.php      --> die Homeland Seite, wo man zufällige Profile von anderen aus demselben Land.
functions.php       --> die wiederholenden Funktionen, die in anderen PHP-Files abgeruft werden können.
index.php           --> die Login Seite.
logout.php          --> Logout-Button-Funktion.
signup.php          --> die Registrierung Seite.
